# 🚀 InterviAI

An AI-powered mock interview platform that helps students and professionals prepare for technical and HR interviews through personalized AI-generated questions, resume analysis, interview history, and performance analytics.

---

## ✨ Features

### 👤 Authentication
- User Registration
- Login & Logout
- JWT Authentication
- Protected Routes

### 📄 Resume
- Resume Upload (PDF)
- Resume Analysis
- Skill Extraction

### 🤖 AI Interviews
- AI-Generated Questions
- Role-Based Interviews
- Difficulty Levels
- Technical & HR Interview Modes

### 📊 Analytics
- Interview History
- Performance Dashboard
- AI Feedback
- Progress Tracking

### 🛠 Admin Dashboard
- User Management
- Interview Monitoring
- Analytics Overview
